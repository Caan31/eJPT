Herramienta para buscar exploits en la base de datos local de Exploit-DB.

• Buscar exploits conocidos  
• Descargar exploits  
• Uso offline

---

## Buscar exploits

```
searchsploit nombre_servicio
```

## Copiar exploit

Descarga el exploit a tu directorio actual.

```
searchsploit -m NUMERO
```

## Mostrar detalles

```
searchsploit -x NUMERO
```
