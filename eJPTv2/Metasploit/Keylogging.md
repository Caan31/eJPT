Permite capturar las teclas que escribe el usuario en el sistema comprometido.

• Captura credenciales  
• Monitorea actividad del usuario  
• Requiere sesión Meterpreter

---

## Iniciar keylogger

Comienza a registrar las pulsaciones de teclado.

```
keyscan_start
```

---

## Ver capturas

Muestra las teclas registradas.

```
keyscan_dump
```

---

## Detener keylogger

Finaliza la captura de teclas.

```
keyscan_stop
```
